Short Overview Of This Project

This website was created to present skills and technology learned in Web Sytems Development class, IS601 at New Jersey Institute of Technology. Author of this page is Diana Zawislak, student of Computer Science Master Certificate.
Technologies used for this project are:

  • GitHub and Git
  • Docker and DockerHub
  • Python and Flask
  • CI/CD
  • Heroku

This project demonstrates ability to use WTForms, Bootstrap Libraries and build various python tests to test functionality of application.
Project contains all requirements of Project 4 as well as features presented in project 2 such as ability to add csv files of songs and locations that allows user to view a map of.
